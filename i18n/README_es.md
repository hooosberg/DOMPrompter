---

**DOMPrompter** es una app de escritorio para macOS que te permite seleccionar elementos visualmente, ajustar propiedades CSS y generar prompts estructurados para IA — para que herramientas como **Cursor**, **Claude Code**, **Codex** y más modifiquen tu código fuente con precisión.

---

## Características

- **Selección precisa de elementos** — Clic para seleccionar, identificación automática de selectores CSS
- **Seguimiento de diferencias de estilo** — Cada cambio registrado como diff antes/después (ej: `width: 200px → 300px`)
- **Etiquetas en lenguaje natural** — Etiqueta elementos con notas para dar contexto a la IA
- **Retroalimentación instantánea** — Botones flotantes modifican el DOM directamente, WYSIWYG
- **Deshacer y rehacer** — Historial completo con `Cmd+Z` / `Cmd+Shift+Z`
- **Local primero** — Sin recopilación de datos, todo permanece en tu dispositivo
