**DOMPrompter**는 페이지 요소를 시각적으로 선택하고, CSS 속성을 미세 조정하고, 구조화된 AI 프롬프트를 생성하는 macOS 데스크톱 앱입니다. **Cursor**, **Claude Code**, **Codex** 등의 AI 도구가 소스 코드를 정확하게 수정할 수 있습니다.

---

## 주요 기능

- **정확한 요소 선택** — 클릭으로 선택, CSS 셀렉터 자동 식별
- **스타일 차이 추적** — 변경 사항을 before/after 차이로 기록 (예: `width: 200px → 300px`)
- **자연어 태그** — 텍스트 태그로 디자인 의도를 AI에 전달
- **즉각적인 피드백** — 플로팅 버튼으로 DOM 직접 수정, WYSIWYG
- **실행 취소/다시 실행** — `Cmd+Z` / `Cmd+Shift+Z`로 전체 작업 이력
- **로컬 우선** — 데이터 수집 없음, 모든 것이 기기에 유지
