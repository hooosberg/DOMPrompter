**DOMPrompter** est une application de bureau macOS qui vous permet de sélectionner visuellement des éléments, d'ajuster les propriétés CSS et de générer des prompts IA structurés — pour que des outils comme **Cursor**, **Claude Code**, **Codex** modifient votre code source avec précision.

---

## Fonctionnalités

- **Sélection précise d'éléments** — Cliquez pour sélectionner, identification automatique des sélecteurs CSS
- **Suivi des différences de style** — Chaque changement enregistré comme diff avant/après (ex : `width: 200px → 300px`)
- **Tags en langage naturel** — Étiquetez les éléments avec des notes pour le contexte IA
- **Retour instantané** — Les boutons flottants modifient le DOM directement, WYSIWYG
- **Annuler et refaire** — Historique complet avec `Cmd+Z` / `Cmd+Shift+Z`
- **Local d'abord** — Aucune collecte de données, tout reste sur votre appareil
