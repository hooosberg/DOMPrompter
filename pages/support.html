البدء السريع

त्वरित प्रारंभ

1

Launch App

Open DOMPrompter, enter the target page URL or launch from your dev server. Auto-connects via Chrome DevTools Protocol.

启动应用

打开 DOMPrompter，输入目标页面 URL 或从开发服务器启动。应用自动通过 Chrome DevTools Protocol 连接页面。

Iniciar la App

Abre DOMPrompter, ingresa la URL de la página objetivo o inicia desde tu servidor de desarrollo. Se conecta automáticamente a través del Chrome DevTools Protocol.

Lancer l'App

Ouvrez DOMPrompter, entrez l'URL de la page cible ou lancez depuis votre serveur de développement. Connexion automatique via le Chrome DevTools Protocol.

App starten

Öffne DOMPrompter, gib die Zielseiten-URL ein oder starte vom Entwicklungsserver. Verbindet sich automatisch über das Chrome DevTools Protocol.

Avvia l'App

Apri DOMPrompter, inserisci l'URL della pagina di destinazione o avvia dal tuo server di sviluppo. Si connette automaticamente tramite Chrome DevTools Protocol.

Iniciar o App

Abra o DOMPrompter, insira a URL da página alvo ou inicie a partir do seu servidor de desenvolvimento. Conecta-se automaticamente via Chrome DevTools Protocol.

アプリを起動

DOMPrompterを開き、ターゲットページのURLを入力するか、開発サーバーから起動します。Chrome DevTools Protocolを介して自動接続します。

앱 실행

DOMPrompter를 열고 대상 페이지 URL을 입력하거나 개발 서버에서 실행합니다. Chrome DevTools Protocol을 통해 자동으로 연결됩니다.

Запуск приложения

Откройте DOMPrompter, введите URL целевой страницы или запустите с сервера разработки. Автоматическое подключение через Chrome DevTools Protocol.

تشغيل التطبيق

افتح DOMPrompter، أدخل عنوان URL للصفحة المستهدفة أو قم بالتشغيل من خادم التطوير. يتصل تلقائيًا عبر Chrome DevTools Protocol.

ऐप लॉन्च करें

DOMPrompter खोलें, लक्ष्य पृष्ठ URL दर्ज करें या अपने डेवलपमेंट सर्वर से लॉन्च करें। Chrome DevTools Protocol के माध्यम से स्वचालित रूप से कनेक्ट होता है।

2

Select Elements

Click any element on the target page. Blue outline highlights the container, floating action buttons appear around the frame.

选择元素

在目标页面点击任意元素。蓝色边框高亮选中容器，浮动动作按钮出现在蓝框周围。

Seleccionar Elementos

Haz clic en cualquier elemento de la página objetivo. El contorno azul resalta el contenedor, y los botones de acción flotantes aparecen alrededor del marco.

Sélectionner des Éléments

Cliquez sur n'importe quel élément de la page cible. Le contour bleu met en surbrillance le conteneur, et des boutons d'action flottants apparaissent autour du cadre.

Elemente auswählen

Klicke auf ein beliebiges Element der Zielseite. Der blaue Rahmen hebt den Container hervor, schwebende Aktionsschaltflächen erscheinen um den Rahmen.

Seleziona Elementi

Fai clic su qualsiasi elemento nella pagina di destinazione. Il contorno blu evidenzia il contenitore, e i pulsanti di azione flottanti appaiono intorno al riquadro.

Selecionar Elementos

Clique em qualquer elemento na página alvo. O contorno azul destaca o contêiner, e os botões de ação flutuantes aparecem ao redor do quadro.

要素を選択

ターゲットページ上の任意の要素をクリックします。青い枠線でコンテナがハイライトされ、フレーム周りにフローティングアクションボタンが表示されます。

요소 선택

대상 페이지에서 아무 요소나 클릭합니다. 파란색 윤곽선이 컨테이너를 강조하고, 프레임 주위에 플로팅 액션 버튼이 나타납니다.

Выбор элементов

Нажмите на любой элемент на целевой странице. Синяя рамка выделяет контейнер, плавающие кнопки действий появляются вокруг рамки.

تحديد العناصر

انقر على أي عنصر في الصفحة المستهدفة. يبرز الإطار الأزرق الحاوية، وتظهر أزرار الإجراءات العائمة حول الإطار.

तत्व चुनें

लक्ष्य पृष्ठ पर किसी भी तत्व पर क्लिक करें। नीली रूपरेखा कंटेनर को हाइलाइट करती है, फ्लोटिंग एक्शन बटन फ्रेम के चारों ओर दिखाई देते हैं।

3

Adjust Properties

Blue W/H buttons adjust size, green for padding, orange for margin. Or enter precise values in the properties panel.

调整属性

蓝色 W/H 按钮调整宽高，绿色按钮调整 padding，橙色按钮调整 margin。也可在右侧属性面板精确输入数值。

Ajustar Propiedades

Los botones azules W/H ajustan el tamaño, los verdes el padding, los naranjas el margen. También puedes ingresar valores precisos en el panel de propiedades.

Ajuster les Propriétés

Les boutons bleus W/H ajustent la taille, les verts le padding, les oranges la marge. Ou saisissez des valeurs précises dans le panneau de propriétés.

Eigenschaften anpassen

Blaue W/H-Schaltflächen passen die Größe an, grüne das Padding, orangene den Margin. Oder gib präzise Werte im Eigenschaftenpanel ein.

Regola Proprietà

I pulsanti blu W/H regolano le dimensioni, quelli verdi il padding, quelli arancioni il margine. Oppure inserisci valori precisi nel pannello proprietà.

Ajustar Propriedades

Os botões azuis W/H ajustam o tamanho, os verdes o padding, os laranjas a margem. Ou insira valores precisos no painel de propriedades.

プロパティを調整

青いW/Hボタンでサイズ調整、緑でパディング、オレンジでマージンを調整します。プロパティパネルで正確な値を入力することもできます。

속성 조정

파란색 W/H 버튼으로 크기 조정, 초록색으로 패딩, 주황색으로 마진을 조정합니다. 속성 패널에서 정확한 값을 입력할 수도 있습니다.

Настройка свойств

Синие кнопки W/H регулируют размер, зелёные — отступы, оранжевые — поля. Или введите точные значения в панели свойств.

ضبط الخصائص

أزرار W/H الزرقاء تضبط الحجم، والخضراء للحشو، والبرتقالية للهامش. أو أدخل قيمًا دقيقة في لوحة الخصائص.

गुण समायोजित करें

नीले W/H बटन आकार समायोजित करते हैं, हरे पैडिंग के लिए, नारंगी मार्जिन के लिए। या गुण पैनल में सटीक मान दर्ज करें।

4

Add Tags

Add text tags describing your change intent. Tag badges attach to the element. Multiple elements can be annotated simultaneously.

添加标签

为元素添加文字标签描述修改意图。标签徽章附着在对应元素上，支持多个元素同时标注。

Agregar Etiquetas

Agrega etiquetas de texto que describan tu intención de cambio. Las insignias de etiqueta se adjuntan al elemento. Se pueden anotar múltiples elementos simultáneamente.

Ajouter des Tags

Ajoutez des tags textuels décrivant vos intentions de modification. Les badges de tags se rattachent à l'élément. Plusieurs éléments peuvent être annotés simultanément.

Tags hinzufügen

Füge Text-Tags hinzu, die deine Änderungsabsicht beschreiben. Tag-Abzeichen werden am Element befestigt. Mehrere Elemente können gleichzeitig annotiert werden.

Aggiungi Tag

Aggiungi tag testuali che descrivono l'intento della modifica. I badge dei tag si attaccano all'elemento. È possibile annotare più elementi contemporaneamente.

Adicionar Tags

Adicione tags de texto descrevendo sua intenção de mudança. Os emblemas de tags se anexam ao elemento. Vários elementos podem ser anotados simultaneamente.

タグを追加

変更意図を説明するテキストタグを追加します。タグバッジは要素に付着します。複数の要素を同時にアノテーションできます。

태그 추가

변경 의도를 설명하는 텍스트 태그를 추가합니다. 태그 배지가 요소에 부착됩니다. 여러 요소를 동시에 주석을 달 수 있습니다.

Добавить теги

Добавьте текстовые теги, описывающие намерение изменения. Значки тегов прикрепляются к элементу. Можно аннотировать несколько элементов одновременно.

إضافة علامات

أضف علامات نصية تصف نية التغيير. تُرفق شارات العلامات بالعنصر. يمكن التعليق على عناصر متعددة في وقت واحد.

टैग जोड़ें

अपने परिवर्तन के इरादे का वर्णन करने वाले टेक्स्ट टैग जोड़ें। टैग बैज तत्व से जुड़ जाते हैं। एक साथ कई तत्वों को एनोटेट किया जा सकता है।

5

Export Prompt

Click export — all selectors + style diffs + tag annotations merged into a structured AI prompt. Copy-paste into Cursor / Claude Code / Codex.

导出提示词

点击导出，所有选择器 + 样式差异 + 标签注释合并为结构化 AI 提示词。复制粘贴到 Cursor / Claude Code / Codex 即可。

Exportar Prompt

Haz clic en exportar — todos los selectores + diferencias de estilo + anotaciones de etiquetas se combinan en un prompt de IA estructurado. Copia y pega en Cursor / Claude Code / Codex.

Exporter le Prompt

Cliquez sur exporter — tous les sélecteurs + différences de style + annotations de tags fusionnés en un prompt IA structuré. Copiez-collez dans Cursor / Claude Code / Codex.

Prompt exportieren

Klicke auf Exportieren — alle Selektoren + Stil-Unterschiede + Tag-Annotationen werden zu einem strukturierten KI-Prompt zusammengeführt. Kopiere und füge in Cursor / Claude Code / Codex ein.

Esporta Prompt

Fai clic su esporta — tutti i selettori + differenze di stile + annotazioni tag vengono uniti in un prompt IA strutturato. Copia e incolla in Cursor / Claude Code / Codex.

Exportar Prompt

Clique em exportar — todos os seletores + diferenças de estilo + anotações de tags são mesclados em um prompt de IA estruturado. Copie e cole no Cursor / Claude Code / Codex.

プロンプトをエクスポート

エクスポートをクリック — すべてのセレクター + スタイル差分 + タグアノテーションが構造化AIプロンプトに統合されます。Cursor / Claude Code / Codex にコピー＆ペーストします。

프롬프트 내보내기

내보내기를 클릭하면 모든 셀렉터 + 스타일 차이 + 태그 주석이 구조화된 AI 프롬프트로 병합됩니다. Cursor / Claude Code / Codex에 복사하여 붙여넣으세요.

Экспорт промпта

Нажмите экспорт — все селекторы + различия стилей + аннотации тегов объединяются в структурированный AI-промпт. Скопируйте и вставьте в Cursor / Claude Code / Codex.

تصدير الأمر

انقر على تصدير — يتم دمج جميع المحددات + فروقات الأنماط + تعليقات العلامات في أمر ذكاء اصطناعي منظم. انسخ والصق في Cursor / Claude Code / Codex.

प्रॉम्प्ट निर्यात करें

निर्यात पर क्लिक करें — सभी सेलेक्टर + स्टाइल अंतर + टैग एनोटेशन एक संरचित AI प्रॉम्प्ट में मर्ज हो जाते हैं। Cursor / Claude Code / Codex में कॉपी-पेस्ट करें।
